# Secure Chat App

## Description
Secure Chat Application is a real-time messaging system built using Python. It uses TCP socket programming to allow multiple users to communicate through a client-server architecture. Users connect using a username instead of an IP address and can send messages in real time through a simple interface.

## Features
- Real-time messaging
- Multi-user chat system
- Client-server architecture
- Username-based communication
- Simple GUI interface

## Technologies Used
- Python
- Socket Programming
- TCP Protocol
- Tkinter

## How to Run

1. Run the server:
python server.py

2. Run the client:
python client.py

3. Enter your username and start chatting.
